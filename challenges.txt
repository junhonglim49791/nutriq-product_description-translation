Why write this?
---------------
- Google sheet appscript and excel translation function will translate the html tags which messed up the description when excel file is uploaded
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Challenges
-----------
Old API
    - API time out
    - Data formatting
    - Version doesnt support exception which doesn't show whats wrong -> upgrade to latest ver.

New API
    - Text nodes that contains alot of commas like ingredients cause translation to be repeated -> need to split in chunks
    - Bulk calls are slower (~1300s) -> use basic (~311s)

Translation accuracy
    The translated results are not that good compare to chatgpt and google translate on web. Its not about how i process before i passed the data in.
    One plain string sentence Example 1:
    Cartilage is the connective tissue that covers the ends of your bones to form a smooth, shock-absorbing surface for your joints (plain string, tested using sub.py)
    ->
        - Rawan adalah tisu penghubung yang meliputi hujung tulang anda untuk *membentuk permukaan yang menghilangkan dan menyerap yang licin* untuk sendi anda 
            (googletrans)
        - Rawan adalah tisu penghubung yang menutupi hujung tulang anda untuk *membentuk permukaan licin, menyerap kejutan untuk sendi anda* 
            (google translate on web)
        - Rawan ialah tisu penghubung yang menutupi hujung tulang anda untuk *membentuk permukaan yang licin dan menyerap hentakan* bagi sendi-sendi anda. (GPT)

    Example 2:
        - Memilih kalsium yang betul adalah *kalsium penting dan elemen adalah kunci* kerana ia adalah jumlah sebenar kalsium badan anda menyerap untuk pertumbuhan 
        tulang dan manfaat kesihatan yang lain. (googletrans)
        - Memilih kalsium yang betul adalah *penting dan unsur kalsium adalah kunci* kerana ia adalah jumlah sebenar kalsium yang diserap oleh badan anda untuk 
        pertumbuhan tulang dan manfaat kesihatan yang lain. (google translate on web)
        - Memilih kalsium yang tepat *adalah penting, dan kalsium elemental merupakan kunci* kerana ia adalah jumlah sebenar kalsium yang diserap oleh tubuh anda 
        untuk pertumbuhan tulang dan manfaat kesihatan yang lain. (chatgpt)
